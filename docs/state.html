bootstrap-nav-step has 4 step to show different condition about the step.

  • 1. Done
  • 2. Active (ongoing)
  • 3. Normal
  • 4. Disabled

Active

Active state used in step which the user currently on.

  • Active state step
<ul class="nav-step">
  <li class="active"><a href="#">Active state step</a></li>
</ul>

Disabled

Disabled state used in step which the user cannot click right now, but we inform what step they will face.

In disabled state, you must not to add <a> tag.

  • Disabled state step
<ul class="nav-step">
  <li class="disabled">Disabled state step</li>
</ul>

Done

Done state used in step which the content is completed by user.

  • Done state step

</ul>
